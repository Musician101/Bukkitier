# Bukkitier

Use Mojang's Brigadier to create commands for your plugins.

```Groovy
repositories {
    mavenCentral()
    maven {
        name = 'GitHubPackages'
        url = 'https://maven.pkg.github.com/Musician101/Bukkitier'
    }
}

dependencies {
    compile 'io.musician101:bukkitier:VERSION'
}
```
