# Bukkitier

Use Mojang's Brigadier to create commands for your plugins.

Groovy:

```Groovy
repositories {
    maven {
        url = 'https://jitpack.io'
    }
}

dependencies {
    implementation 'com.github.Musician101:Bukkitier:[version]'
}
```

Kotlin DSL

```Kotlin
repositories {
    maven("https://jitpack.io")
}

dependencies {
    api("com.github.Musician101:Bukkitier:[version]")
}
```
