# Bukkitier

Use Mojang's Brigadier to create commands for your plugins.

Groovy:
```Groovy
repositories {
    maven {
        url = 'https://jitpack.io'
    }
}

dependencies {
    implementation 'com.github.musician101:bukkitier:1.2'
}
```

Kotlin DSL
```Kotlin
repositories {
    maven("https://jitpack.io")
}

dependencies {
    api("com.github.musician101:bukkitier:1.2")
}
```
